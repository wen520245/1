# 华商创新班成绩管理系统

> 专为23本计算机科学与技术（创新实验班）【1-2】班设计的成绩管理平台

## 📋 项目简介

本系统是一款基于HTML + CSS + JavaScript技术栈开发的成绩管理系统，采用Chart.js实现数据可视化，涵盖计算机前沿技术课程的成绩管理，包括讨论、作业、签到、课程积分、分组任务等多个评分维度。

## ✨ 功能模块

| 模块 | 功能描述 |
|------|----------|
| 📊 数据概览 | 实时展示学生总数、课程数量、成绩记录、及格人数等核心指标，包含多个可视化图表 |
| 👥 学生管理 | 学生信息的增删改查，支持搜索筛选，管理学生状态 |
| 📚 课程管理 | 课程信息管理，包括学分、学时、授课教师、开课学期等 |
| 📝 成绩管理 | 成绩录入、编辑、查询，支持按课程筛选，包含讨论、作业、签到等多个评分维度 |
| 🎯 数据分析 | 成绩趋势分析、性别成绩对比、各科成绩对比、分数段分布统计 |

## 🚀 快速开始

### 本地预览

直接在浏览器中打开 `index.html` 即可预览系统，无需任何依赖安装。

### 部署到服务器

#### 使用 Nginx

```nginx
server {
    listen 80;
    server_name your-domain.com;
    
    root /path/to/project;
    index index.html;
    
    location / {
        try_files $uri $uri/ =404;
    }
}
```

#### 使用 Python SimpleHTTPServer

```bash
python -m http.server 8000
```
